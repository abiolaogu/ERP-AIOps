## Formula Logic

- Revenue = Subscription + Usage + Services
- Gross Profit = Revenue x Gross Margin
- EBITDA = Gross Profit - Opex
- Free Cash Flow = EBITDA - Capex - Working Capital Delta
- LTV = (ACV x Gross Margin) / Logo Churn
- CAC Payback (months) = CAC / ((ACV x Gross Margin) / 12)
- MOIC = Investor Proceeds / Invested Capital
- IRR = (MOIC^(1/5)) - 1
