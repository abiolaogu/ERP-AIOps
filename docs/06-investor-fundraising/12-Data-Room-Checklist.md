## 10. Data Room Access Protocol

1. All investors must execute a mutual NDA before VDR access is granted
2. VDR access is view-only; no downloads without explicit permission
3. Watermarking enabled on all documents (investor name + date)
4. Access is logged and audited (who viewed what, when)
5. Access expires 90 days after term sheet execution or upon deal termination
6. Questions routed through designated contact (no direct team outreach without permission)
