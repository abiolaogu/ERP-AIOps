## 4. Proposed Deal Structure

### 4.1 Key Terms

| Term | Detail |
|---|---|
| **Security** | Series A Preferred Stock |
| **Liquidation preference** | 1x non-participating preferred |
| **Anti-dilution** | Broad-based weighted average |
| **Board composition** | 5 seats: 2 founders, 2 investors, 1 independent |
| **Protective provisions** | Standard (sale, new equity, debt >$1M, budget changes >20%) |

| **Pro-rata rights** | Series A investors have pro-rata rights in future rounds |
| **Drag-along** | Majority of preferred + majority of common |
